Music Recommendation System using Real-Time Data

  • Tech Stack: Python, Apache Kafra, PySpark, Spark, Docker
  • Github URL: Project Link

The objective of this project is to provide personalized music recommendations in real time for Spotify users. The primary goal is to enhance user experience by leveraging real-time data and machine learning models to suggest songs dynamically based on user preferences.

This recommendation system processes streaming music data, offering users highly relevant song suggestions in real time, making use of distributed big data technologies such as PySpark, Spark Streaming, and Kafka. The project also focuses on scalability and efficiency through the use of Docker to containerize and manage services seamlessly.

Techniques:

  • Real-time data processing using Spark Streaming and Kafka
  • Unsupervised machine learning for recommendation
  • Distributed data processing with PySpark
  • Deployment and management of services using Docker

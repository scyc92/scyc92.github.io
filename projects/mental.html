Mental Health Prediction Using Machine Learning

  • Tech Stack: Python, XGBoost, GridSearchCV, Pandas, Seaborn
  • Github URL: Project Link
  • Kaggle: Kaggle Link

This project focuses on predicting mental health outcomes, specifically depression, using machine learning models trained on survey data. The goal is to leverage advanced techniques to provide accurate and efficient predictions that can be used in research or real-world scenarios to better understand the factors contributing to mental health challenges.

The data is preprocessed, explored, and modeled using techniques like one-hot encoding, label encoding, and missing value imputation. The XGBoost classifier is optimized using GridSearchCV to achieve the best performance. The project also emphasizes data visualization to gain insights into the features affecting mental health outcomes.

Techniques:

  • Data preprocessing, including encoding categorical variables and handling missing values
  • Exploratory Data Analysis (EDA) with Seaborn for data insights
  • Hyperparameter tuning using GridSearchCV for improved model accuracy
  • Deployment-ready code for generating predictions and preparing submission files
